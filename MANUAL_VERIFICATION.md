# 手动验证指南

本指南概述了手动验证 Electron 答题应用功能的步骤。

## 准备工作

1.  **Node.js**: 确保已安装 Node.js (v18+)。
2.  **依赖**: 在项目目录中运行 `npm install`。
3.  **构建**: 运行 `npm run build` 以验证生产环境构建。
4.  **运行**: 使用 `npm run dev` 在开发模式下启动应用。

## 测试场景

### 1. 题库导入
1.  启动应用程序。
2.  在首页点击“选择文件 (Select File)”。
3.  选择一个包含题目的有效 Excel 文件 (`.xlsx`)，文件应包含以下列：`题干`, `选项A`, `选项B`, `参考答案`, `分类`, `题型`, `注释`, `难度`。
4.  **验证**:
    *   显示成功消息，并显示导入的题目数量。
    *   仪表板上的“总题目数 (Total Questions)”更新。
    *   “分类练习 (Topic Practice)”下拉菜单中加载了分类信息。

### 2. 练习模式

#### 顺序练习 (Sequential Practice)
1.  在首页点击“顺序练习 (Sequential Practice)”。
2.  **验证**:
    *   题目按顺序显示。
    *   点击选项会显示即时反馈（正确显示绿色，错误显示红色）。
    *   “上一题 (Previous)”和“下一题 (Next)”按钮导航正常。
    *   导航回已回答的题目会显示选择的答案和结果。

#### 随机练习 (Random Practice)
1.  在首页点击“随机练习 (Random Practice)”。
2.  **验证**:
    *   题目以随机顺序显示。
    *   功能与顺序练习相同。

#### 分类练习 (Topic Practice)
1.  在首页的下拉菜单中选择一个分类。
2.  点击“开始分类练习 (Start Topic)”。
3.  **验证**:
    *   仅显示属于所选分类的题目。

#### 错题回顾 (Mistake Review)
1.  在其他模式下回答错误一些题目。
2.  返回首页并点击“错题回顾 (Mistake Review)”。
3.  **验证**:
    *   仅显示之前回答错误的题目。

#### 收藏练习 (Favorites)
1.  在任何测验模式下，点击“星形 (Star)”图标将题目标记为收藏。
2.  返回首页并点击“收藏练习 (Favorites)”。
3.  **验证**:
    *   仅显示已标记的题目。
    *   取消标记题目后（刷新/重新进入后），该题目会从列表中移除。

### 3. 测验内功能

#### AI 解析 (AI Explanation)
1.  在“设置 (Settings)”中，配置有效的 API Key（如果使用 ZhiPu 等自定义提供商，请配置 Base URL）。
2.  在测验中，点击“请求 AI 解析 (Ask AI for explanation)”。
3.  **验证**:
    *   按钮显示加载状态。
    *   几秒钟后，题目下方会出现解析内容。

#### 笔记 (Notes)
1.  点击题目右上角的“消息/笔记 (Message/Note)”图标。
2.  输入笔记并点击“保存笔记 (Save Note)”。
3.  离开并返回该题目。
4.  **验证**:
    *   笔记已持久化保存并显示。

### 4. 搜索 (Search)
1.  在首页点击“搜索题目 (Search Questions)”。
2.  输入一些题目中存在的关键词。
3.  点击“搜索 (Search)”。
4.  **验证**:
    *   出现匹配的题目列表。
    *   点击结果会展开显示详情和正确答案。

### 5. 模拟考试 (Mock Exam)
1.  在首页点击“模拟考试 (Mock Exam)”。
2.  回答题目。注意此时不会给出即时反馈。
3.  在最后一题点击“提交试卷 (Submit Exam)”。
4.  **验证**:
    *   出现确认对话框。
    *   确认后，显示最终得分。
    *   回顾题目现在会显示正确/错误状态。

### 6. 统计 (Statistics)
1.  通过侧边栏进入“结果 (Results)”页面。
2.  **验证**:
    *   总题目数、正确答案数、错题数和收藏数准确无误。
    *   进度条反映当前状态。

### 7. 设置 (Settings)
1.  进入“设置 (Settings)”页面。
2.  更新 AI 配置。
3.  点击“保存 (Save)”。
4.  重启应用。
5.  **验证**:
    *   设置已持久化保存。

## 故障排除

*   **数据库错误**: 如果遇到数据库问题，尝试删除用户数据目录中的 `quiz-app.db` 文件（开发过程中会在控制台打印路径）以重置。
*   **Excel 解析**: 确保 Excel 文件表头完全匹配（例如 "题干", "参考答案"）。
